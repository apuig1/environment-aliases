# Environment Aliases for BASH

```
vim ~/.bash_profile
```

Refresh aliases without rebooting:

```
source ~/.bash_profile
```

# Environment Aliases for ZSH

```
vim ~/.zshrc
```

Refresh aliases without rebooting:

```
source ~/.zshrc
```
